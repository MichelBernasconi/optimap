# Security Policy

## Supported Versions

Optimap supports the following versions of the extension:

| Version | Supported          |
| ------- | ------------------ |
| v0.0.1+ | :white_check_mark: |
| < v0.0.1| :x:                |

## Reporting a Vulnerability

If you've found a vulnerability or a security issue in OptiMap, we take it very seriously. Please follow these steps:

1.  **Do Not Create a Public Issue**: To prevent a vulnerability from being exploited before it's fixed, we ask you not to post security issues in public issue trackers.
2. **Contact Us Directly**: Reach out via GitHub Private Vulnerability Reporting or contact the maintainers at 
   [dannydoodlesstory@gmail.com](mailto:dannydoodlesstory@gmail.com).
3.  **Provide Details**: Include as much information as possible:
    *   Description of the impact.
    *   Step-by-step instructions to reproduce.
    *   Environment details (OS version, VS Code version).

We will acknowledge your report within 48-72 hours and provide a timeline for the resolution.
