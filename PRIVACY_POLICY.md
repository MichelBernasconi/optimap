# Privacy Policy for OptiMap

OptiMap is committed to protecting your privacy. This policy outlines how we handle data and respect your confidentiality within the VS Code environment.

## Data Processing

### 1. Local-Only Analysis
All file analysis performed by OptiMap occurs exclusively on your local machine. No source code or workspace data is uploaded to our servers or third-party platforms during the analysis phase.

### 2. AI Delegation
If you choose to use the "Apply AI Fix" feature, OptiMap facilitates a local command (`optimap.delegateToAgent`) that interacts with an AI agent. Any interaction with AI models happens within your own VS Code environment and is subject to the privacy policies of the AI provider you are using (e.g., Azure OpenAI, Claude, etc.).

### 3. Telemetry
OptiMap does **not** collect telemetry data, usage statistics, or crash reports by default. We respect your "No Telemetry" preference if configured in VS Code.

## External Resources
OptiMap uses D3.js and other libraries for visualization. These are loaded from public CDNs within the sidebar webview. No session-specific data is shared with these CDNs.

## Changes to this Policy
Any updates to this policy will be communicated via subsequent release notes on the VS Code Marketplace.
